# Calendario
Calendário Web, criado em 2020 como projeto final no curso CS50, de introdução à programação em Python, desenvolvido por Harvard e ofertado na plataforma edX. Desenvolvido em python, com o framework web Flask, HTML, CSS e muita ajuda do bootstrap. Bando de dados em SQL, que armazena informações de login e senha (criptografada), bem como eventos e lembretes criados pela plataforma.
